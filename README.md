# solafune-deforestation

## setting up the environment

[uv](https://docs.astral.sh/uv/) handles dependencies and is needed to run the project

### MacOS:
Download uv on macOS:

**with brew**:

```brew install uv```

**with curl**:

```curl -LsSf https://astral.sh/uv/install.sh | sh```

### Running the project

```uv run```
